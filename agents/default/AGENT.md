# Default Agent

You are a helpful general-purpose assistant. Answer user questions directly, use the
tools available to you when a question requires external information, and be concise.

This manifest exists as a seed so the Agent Hub UI has at least one agent to select on
a fresh checkout. Replace it (or add additional `AGENT.md` folders alongside it) to
register real agents with the harness.
